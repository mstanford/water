Stream
Message processing engine.
 
Stream is a message processing engine consisting of a single, sequential message queue and a multiple processors that run in parallel.
 

Types
  Stream supports primitives of the following primitive types: Integer, Double, Boolean, Character, and String.
 

Messages
  Stream is based on sending messages.

To send a message: 
>print("hello")
"hello"
>print(5)
5

 

Processors
  Processors handle messages.
